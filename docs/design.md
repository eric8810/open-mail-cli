
## 11. 部署和发布

### 11.1 打包方式
- npm包发布
- 独立可执行文件（pkg）
- Docker镜像（可选）

### 11.2 安装方式
```bash
# 通过npm全局安装
npm install -g mail-client

# 通过源码安装
git clone https://github.com/username/mail-client.git
cd mail-client
npm install
npm link
```

### 11.3 版本管理
- 遵循语义化版本（Semantic Versioning）
- 主版本号.次版本号.修订号
- 例如：1.0.0, 1.1.0, 1.1.1
